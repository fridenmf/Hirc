type InternalChan = Chan (Target, String)Source

Used for internal commnands like Logging (Log). Don't use this directly, pass it as an argument.

type ExternalChan = Chan EventSource

Used for external commands, like sending a private message (PvtMsg). Use helper functions instead of using this directly, for example privateMessage.

data Event Source

Events from the server. Patternmatch this in your BotThread.
